This repository contains a **Star Wars ship catalog app** using React framework. The application serves as a Star Wars starship catalog based on the information of the Star Wars API (https://swapi.dev). It gathers the starships information collection of the Star Wars cinematographic world displaying several key information of the starship, the films in which the starship appears and its pilots. The application is made using Redux library for global state management and the information is obtained via API calls with axios.

- **Testing:** React-testing library and vitest are used to test the correct functionalities of three components reproducing the actions and store of the application.
